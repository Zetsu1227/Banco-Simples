# Banco Simples
Programa de gerenciamento de dados e contas -- POO

1) Conta

	+ O usuário é capaz de manipular os valores de sua conta por meio de saques e depósitos.
	+ Essas ações são guardados no histórico da conta.
	+ É possível serializar uma conta, isto é, guardar os seus dados em uma string -> ATUALIZAR: ESCREVER EM .txt
	+ E há o processo inverso, de criar uma conta a partir dessa string -> ATUALIZAR: LER .txt 

	->Informações de uma conta:
		a) Numero 
		b) Titular
		c) Saldo
		d) Histórico
